# คำอธิบายโปรแกรม ShippingCalculator.java

## บทนำ
โปรแกรม ShippingCalculator.java เป็นโปรแกรมคำนวณค่าจัดส่งสินค้าที่พัฒนาด้วยภาษา Java โดยโปรแกรมจะรับข้อมูลจากผู้ใช้งาน เช่น ปลายทาง น้ำหนักสินค้า บริการเสริม และสถานะสมาชิก เพื่อคำนวณค่าจัดส่งที่ถูกต้องตามเงื่อนไขที่กำหนด

## โครงสร้างโปรแกรม

### ค่าคงที่ (Constants)
โปรแกรมมีการกำหนดค่าคงที่สำหรับการคำนวณค่าจัดส่ง ได้แก่:

- **ค่าจัดส่งสำหรับพื้นที่กรุงเทพฯ และปริมณฑล**:
  - น้ำหนัก 0-1 กก.: 40 บาท
  - น้ำหนัก 1-3 กก.: 60 บาท
  - น้ำหนัก 3-5 กก.: 80 บาท
  - น้ำหนักมากกว่า 5 กก.: 100 บาท

- **ค่าจัดส่งสำหรับต่างจังหวัด**:
  - น้ำหนัก 0-1 กก.: 60 บาท
  - น้ำหนัก 1-3 กก.: 90 บาท
  - น้ำหนัก 3-5 กก.: 120 บาท
  - น้ำหนักมากกว่า 5 กก.: 150 บาท

- **ค่าบริการเพิ่มเติม**:
  - ค่าบริการด่วน: 30 บาท
  - อัตราส่วนลดสมาชิก VIP: 20%

## ขั้นตอนการทำงานของโปรแกรม

### 1. การเริ่มต้นโปรแกรม
- โปรแกรมเริ่มทำงานที่เมทอด main()
- สร้างอ็อบเจกต์ Scanner เพื่อรับข้อมูลจากผู้ใช้

### 2. แสดงข้อความต้อนรับ
- แสดงข้อความ "=== ระบบคำนวณค่าจัดส่งสินค้า ==="
- แสดงคำแนะนำในการเลือกปลายทาง

### 3. รับข้อมูลปลายทาง
- แสดงตัวเลือก 1 สำหรับ "กรุงเทพฯ และปริมณฑล" และ 2 สำหรับ "ต่างจังหวัด"
- รับค่าตัวเลขจากผู้ใช้และแปลงเป็นค่า boolean (isBangkokArea)

### 4. รับน้ำหนักสินค้า
- รับค่าน้ำหนักจากผู้ใช้
- ตรวจสอบว่าน้ำหนักต้องมากกว่า 0 กิโลกรัม หากไม่ตรงตามเงื่อนไขจะแสดงข้อความแจ้งเตือนและออกจากโปรแกรม

### 5. รับข้อมูลบริการด่วน
- รับค่า 1 ถ้าต้องการบริการด่วน หรือ 0 ถ้าไม่ต้องการ
- เก็บค่าในตัวแปร boolean (isExpress)

### 6. รับข้อมูลสถานะสมาชิก VIP
- รับค่า 1 ถ้าเป็นสมาชิก VIP หรือ 0 ถ้าไม่ใช่
- เก็บค่าในตัวแปร boolean (isVIP)

### 7. คำนวณค่าจัดส่ง
- เรียกใช้เมทอด calculateBaseCost() เพื่อคำนวณค่าจัดส่งพื้นฐานตามปลายทางและน้ำหนัก
- คำนวณค่าบริการด่วน (ถ้าเลือก)
- คำนวณราคารวมก่อนหักส่วนลด
- คำนวณส่วนลดสำหรับสมาชิก VIP
- คำนวณราคารวมสุทธิ

### 8. แสดงผลลัพธ์
- แสดงรายละเอียดค่าจัดส่ง ได้แก่:
  - ปลายทาง
  - น้ำหนักสินค้า
  - ค่าขนส่งพื้นฐาน
  - ค่าบริการด่วน (ถ้ามี)
  - ราคารวมก่อนหักส่วนลด
  - ส่วนลดสมาชิก VIP (ถ้ามี)
  - ยอดชำระสุทธิ

### 9. ปิดโปรแกรม
- ปิดอ็อบเจกต์ Scanner
- จบการทำงานของโปรแกรม

## เมทอด calculateBaseCost
เมทอดนี้ใช้สำหรับคำนวณค่าจัดส่งพื้นฐานตามเงื่อนไข:
- ตรวจสอบว่าเป็นพื้นที่กรุงเทพฯ หรือต่างจังหวัด
- ตรวจสอบช่วงน้ำหนักของสินค้า
- คืนค่าค่าจัดส่งตามตารางที่กำหนดไว้

## การจัดการข้อผิดพลาด
- โปรแกรมตรวจสอบค่าน้ำหนักที่ผู้ใช้ป้อนว่าต้องมากกว่า 0
- หากน้ำหนักไม่ถูกต้อง โปรแกรมจะแสดงข้อความแจ้งเตือนและออกจากโปรแกรมทันที

## ประโยชน์ของการใช้โปรแกรม
- ช่วยคำนวณค่าจัดส่งได้อย่างแม่นยำตามเงื่อนไขต่างๆ
- สะดวกและรวดเร็วในการใช้งาน
- แสดงรายละเอียดค่าใช้จ่ายอย่างชัดเจน
- รองรับการคำนวณส่วนลดสำหรับสมาชิก VIP
